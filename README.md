```bash

```
## 4. Authenticatie & Security
```bash
npm install @nestjs/jwt @nestjs/passport passport-jwt bcrypt helmet
```

## 5. Validatie & Transformatie
```bash
npm install class-validator class-transformer
```

```bash
npm install multer
```
## 7. E-mail Services
```bash
npm install @sendgrid/mail nodemailer @nestjs-modules/mailer
```

## run applicatie localhost
